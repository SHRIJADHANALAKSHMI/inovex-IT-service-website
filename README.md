# Overview

**Inovex** is a modern SaaS-style project management and client collaboration platform built using the **MERN Stack**.

The platform enables businesses to:

✅ manage client projects
✅ track development workflow
✅ communicate in real-time
✅ upload project deliverables
✅ monitor project progress
✅ provide premium client experience

Inspired by platforms like:

- Jira
- ClickUp
- Fiverr
- Slack
- Monday.com
- Upwork

# ⚙️ Installation & Setup

# 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/inovex-saas.git
```

---

# 2️⃣ Backend Setup

```bash
cd backend
npm install
npm start
```

---

# 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create `.env` inside backend:

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
PORT=5000
```
